# Twitch Chat Simulator

A simple Python script that simulates a live Twitch chat. I made this for unpopular streamers who want to feel like they actually have viewers, or just for shits and giggles.

It randomizes thousands of potential username combinations, gives them colored text in the terminal, and spits out classic chat brainrot/copypastas at random intervals so it looks like a real stream.

How to use it
Make sure you have Python installed.

Download twitchChatSim.py

Open your terminal and run:

python main.py

Press enter as prompted in terminal and enjoy the fake chat

Feel free to open a pull request if you want to add more stupid phrases to the word list.
